### 🎯 What I Actually Do

> I help businesses find the work that shouldn't need humans — then automate it with AI.

My engagements start with a **workflow audit**: mapping what your team does, finding the bottlenecks, and identifying where AI creates the most leverage.

Then I build it — using **n8n**, **Make.com**, and **LLM APIs** to design automation pipelines that handle repetitive cognitive work at scale, with error handling, monitoring, and fallback logic from day one.

When the workflow needs custom software to make it permanent, I build that too — full-stack React and Node.js applications that are production-grade and actually maintainable.

**The result:** teams that do more, workflows that compound, software that ships.
